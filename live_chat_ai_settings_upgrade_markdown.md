# LIVE CHAT AI SYSTEM UPGRADE
## Struktur Pengaturan Baru untuk AI Customer Service Universal

---

# TUJUAN PEMBARUAN

Sistem Live Chat AI diperbarui agar:

- lebih mudah digunakan
- cocok untuk semua jenis bisnis
- tidak membutuhkan kemampuan teknis tinggi
- AI terasa seperti Customer Service manusia
- lebih fokus conversion
- lebih mudah diatur owner/admin
- mendukung automation & lead management

# 3. CHAT FLOW

## Tujuan
Mengatur alur percakapan AI.

## Tambahkan Visual Flow Builder

```text
Greeting
↓
Pilih Kebutuhan
↓
Gali Masalah
↓
Berikan Solusi
↓
CTA
↓
Ambil Nama
↓
Ambil WA
↓
Transfer Admin
```

---

## Fitur Wajib

### Greeting Message

Admin dapat:
- edit greeting
- tambah variasi greeting
- setting greeting berdasarkan jam

---

### Tombol Cepat

Admin dapat:
- tambah tombol
- hapus tombol
- drag and drop urutan
- ubah icon
- ubah response

Contoh:

```text
📦 Produk/Jasa
💰 Harga
📚 Konsultasi
📅 Booking
👨‍💼 Admin
❓ FAQ
```

---

### Node Flow Builder

Contoh:

```text
Jika klik "Harga"
→ tampilkan pricing

Jika klik "Konsultasi"
→ tanya kebutuhan

Jika klik "Admin"
→ minta nama
```
